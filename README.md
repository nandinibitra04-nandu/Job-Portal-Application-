# Job Portal Web Application

## Project Overview
The Job Portal Web Application is a full-stack web application developed to simplify the job search and recruitment process. The application enables users to register, log in securely, browse available job listings, and apply for jobs through a user-friendly interface.

## Features
- User Registration and Login
- JWT-based Authentication
- Password Encryption using bcrypt
- Browse Available Job Listings
- Apply for Jobs
- RESTful API Integration
- MongoDB Database Connectivity

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcrypt

## Installation

1. Clone the repository

```bash
git clone https://github.com/nandinibitra04-nandu/Job-Portal.git
```

2. Install dependencies

```bash
npm install
```

3. Start the server

```bash
node server.js
```
